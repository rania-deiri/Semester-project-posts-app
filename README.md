## Getting Started

### Installing

1. Clone the repo:

```bash
git clone (https://github.com/rania-deiri/Semester-project-posts-app.git)
```
2. Install xampp
3. Place project in htdocs inside xampp path
4. Open Project using vs code
5. Run npm install
6. Run npm start
7. Login with
